Projects Overview

This is the landing page for planned projects that the technology club will take on. These projects will be technology related things that we can do as a group to get some hands on experience completed projects. They can be anything from building a computer to giveaway, building a computer for display, making a web page for the school like a club directory, or even something like developing a program (like a game) as a group. This section will also contain information about any guest speakers we plan to have, so please look forward to that!
